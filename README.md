# epubcif

> [EPUB Canonical Fragment Identifiers 1.1](https://idpf.org/epub/linking/cfi/epub-cfi.html)

## Setup dev env

first time.

```shell
$ conda create --prefix ./.venv python=3.12.7 -y
$ conda activate ./.venv
$ pip install --upgrade pip
$ pip install -r requirements.txt
$ ./patches/patch_mac.sh
```

after open terminal.

```shell
$ conda activate ./.venv
```

exit env.

```shell
conda deactivate
```
